Reflection

Aspect Details
HTML
  • I referred a lot to lecture notes & online resources such as w3schools to make sure I got the HTML code correctly.
  • I had to test every change of HTML code multiple times so that I could match the results of the code to my design.
Design & CSS
  • I researched several design ideas on dribbble.com and other places to get some inspiration. Some designs that I looked at include Chaitanya's, Abo's and also Farhan's.
  • I decided to use the free open source CSS framework, Bootstrap because it will enable me to rapidly develop my website while also allow me to customise some parts of my website to my liking.
  • I even decided to use Bootstrap's logo in my website because it goes with my name, Balqis, and with the colour purple which I decided to have as my primary colour. As for its license, I made sure that I'm not violating any copyrights.
  • I also had to test every change of CSS code multiple times so that I could match the results of the code to my design.
